# iTerm2 Puppet Module for Boxen
[![Build+Status](https://travis-ci.org/boxen/puppet-iterm2.png?branch=master)](https://travis-ci.org/boxen/puppet-iterm2)


## Usage

```puppet
# Stable release
include iterm2::stable

# Dev release
include iterm2::dev
```

## Required Puppet Modules

* boxen
* stdlib
